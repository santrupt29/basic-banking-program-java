# Basic Banking Program (Java)

A simple console-based banking application written in Java.

## Features
- Show Balance
- Deposit Money
- Withdraw Money
- Exit Program

## Concepts Used
- Java Scanner
- while loop
- switch-case
- methods
- basic OOP structure

## How to Run
1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Run `Main.java`

Made while learning Java
